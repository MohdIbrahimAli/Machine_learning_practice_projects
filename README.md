# Machine learning practice projects
Here you can find nearly **20 projects** for implementation of Machine learning and Deep Learning concepts(beginner level)

By practicing these problems you can have a pretty good understanding of the basic concepts of the ML and DL to get started in any project of your own.   

Every folder contains there own detailed README.md. Read them carefully before running the program.

Every README.md contains deep and in depth knowledge for all the concepts used in the program to get the fundamentals correct and have a better understanding of the concepts stated in them.

_for people trying to learn from here it is advised to practice more on their own after getting some of the understanding for the concepts used here to get the most out of this._
Hard work pays best if it is done consistently over long period of time.
